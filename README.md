# Negamam Silks Website

Commercial landing page for **Negamam Silks in Palladam**, designed as a premium single-page website for showcasing silk saree collections, local trust, and lead-generation calls to action.

## Files
- `index.html` — main marketing website page structure.
- `styles.css` — standalone stylesheet for the website design.
- `google782c2e9acc06b349.html` — Google site verification file.

## Preview
Open `index.html` in a browser or deploy it via any static hosting platform.
